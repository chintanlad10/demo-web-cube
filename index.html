Rubik's Cube Simulator

Interactive 3D Rubik's Cube with built-in solver. Control the cube using keyboard or mouse, then see it solved quickly with efficient C++ code.

Main Controls

Key Combination Action Description
U, D, F, B, R, L Face Rotations Rotate cube faces clockwise
shift + Face Key Counter-clockwise Rotate faces counter-clockwise
return Solve Automatically solve the cube
Esc Reset Reset cube to solved state
X, Y, Z Rotate View Rotate the cube view around respective axes
Scroll Wheel Zoom Zoom in/out of the cube
